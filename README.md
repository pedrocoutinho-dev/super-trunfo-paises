# 🃏 Desafio Super Trunfo - Países (Tema 2)

Este projeto é um simulador do jogo **Super Trunfo**, desenvolvido em **C**. Nesta etapa, o sistema permite o cadastro de duas cartas de cidades e realiza a comparação automática de atributos para determinar a vencedora.

## 🚀 Funcionalidades
- **Cadastro Duplo**: Entrada de dados para duas cidades diferentes.
- **Comparação por Atributos**: O sistema compara automaticamente a **População** e o **PIB**.
- **Lógica de Decisão**: Uso de estruturas `if-else` para determinar vitória ou empate.
- **Formatação de Dados**: Exibição limpa de valores monetários e numéricos.

## 🛠️ Como Compilar e Rodar
1. Tenha um compilador de C instalado (GCC ou Visual Studio Code).
2. No terminal, use:
   ```bash
   gcc super_trunfo.c -o super_trunfo
   ./super_trunfo
